# Contributing to NexuzyAI

Thank you for your interest in contributing! NexuzyAI is fully open source and welcomes contributions of all kinds.

---

## Ways to Contribute

- 🐛 **Bug reports** — Open an [issue](https://github.com/david0154/NexuzyAI-Android/issues)
- ✨ **Feature requests** — Open an issue with the `enhancement` label
- 🔧 **Code contributions** — Submit a pull request
- 🌐 **Translations** — Add new locale support
- 📝 **Documentation** — Improve README or SETUP_GUIDE

---

## Getting Started

```bash
# 1. Fork the repo on GitHub
# 2. Clone your fork
git clone https://github.com/YOUR_USERNAME/NexuzyAI-Android.git

# 3. Create a feature branch
git checkout -b feature/my-feature

# 4. Make your changes
# 5. Test on a real Android device (minSdk 26)

# 6. Commit with a clear message
git commit -m "feat: add XYZ feature"

# 7. Push and open a Pull Request
git push origin feature/my-feature
```

---

## Code Style

- Kotlin idioms preferred
- Follow existing package structure (`llm/`, `tools/`, `middleware/`, `adapter/`)
- No hardcoded API keys — use `BuildConfig` or `local.properties`
- No data collection of any kind

---

## Pull Request Guidelines

- Keep PRs focused on one thing
- Include a description of what changed and why
- Test on Android API 26+ before submitting
- Reference related issues with `Fixes #123`
